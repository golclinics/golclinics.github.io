Data Structures and Algorithms
— from zero to hero.

A program to help students from all walks of life to level up on
data structures, algorithms, system design and other Computer Science Fundamentals.

The training will be a 3 month program; however, there will be an open community that will be running meetups, chat forums, mock-interviews, break-out sessions and many more — indefinitely

Core Topics

Level 1 - Foundations: Arrays and Strings, Stacks and Queues I, Big-O Analysis, Searching, Sorting I

Level 2 - Intermediate: Linked Lists, Stacks and Queues II, Dictionaries/Hashmaps, Sets/Hashsets, Recursion, Sorting II

Level 3- Advanced: Graphs, Trees, Greedy Algorithms, Backtracking, Dynamic Programming

Timelines

Pilot 0 done

We ran a small pilot in Feb/March, covering parts of Level 1.

Pilot 1 current

The main pilot kicked off in May 2021. Recordings for all the sessions are available in Teams.

Sessions run every Tuesday and Friday, 7.30 - 9.00 am, EAT.
